# my-blog-back-app
## Бэкенд веб-приложения блога
### Реализованы следующие endpoint:
* Получение списка постов (на главной странице — ленте постов) — метод GET: /api/posts?search=Lalala&pageNumber=1&pageSize=5
* Получение поста (на странице поста) — метод POST: /api/posts/1
* Добавление поста (на главной странице — ленте постов) — метод POST: /api/posts
* Редактирование поста (на странице поста) — метод PUT: /api/posts/3
* Удаление поста (на странице поста) — метод DELETE: /api/posts/1
* Инкремент числа лайков поста (на странице поста) — метод POST: /api/posts/1/likes
* Обновление картинки поста (при добавлении и редактировании поста) — метод PUT: /api/posts/1/image
* Получение картинки поста (главная страница — лента постов и страница поста) — метод GET: /api/posts/1/image
* Получение комментариев поста (на странице поста) — метод GET: /api/posts/1/comments
* Получение комментария поста (на странице поста) — метод GET: /api/posts/1/comments/2
* Добавление комментария к посту (на странице поста) — метод POST: /api/posts/1/comments
* Редактирование комментария к посту (страница поста) — метод PUT: /api/posts/1/comments/2
* Удаление комментария (страница поста) — метод DELETE: /api/posts/1/comments/2
